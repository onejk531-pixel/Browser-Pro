# Via Advanced Browser

A lightweight, privacy-focused Android browser with advanced features.

## Features

- **Lightweight & Fast**: Small footprint, quick startup
- **Privacy Focused**: Built-in ad blocking, tracker protection
- **Custom DNS**: Support for custom DNS servers and DNS-over-HTTPS
- **Built-in VPN**: Integrated VPN functionality
- **Script Injection**: Custom user scripts support
- **Certificate Pinning**: Enhanced security with certificate pinning
- **HTTPS Everywhere**: Automatic HTTPS upgrades
- **Password Manager**: Secure password storage and auto-fill
- **Reading Mode**: Distraction-free reading
- **Video Downloader**: Built-in media download capability
- **QR Scanner**: Quick URL scanning
- **Text Reflow**: Better reading experience

## Installation

1. Download and install Android Studio
2. Open the project in Android Studio
3. Build and run on your device/emulator
